# Lab Report Interpretation

## Method

- 先区分报告类型、指标名称、数值、单位和参考范围。
- 结合年龄、性别、症状、基础病和用药背景解释指标意义。
- 指标解释要避免孤立判断，强调趋势、复查和临床背景。
- 若报告涉及急危重异常或症状伴随异常，应优先做风险评估。

## Red Lines

- 不根据单个指标直接诊断疾病。
- 不忽略单位和参考范围。
- 不把检查报告解释成治疗处方。
