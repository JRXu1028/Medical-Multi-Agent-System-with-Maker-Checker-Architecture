# Symptom Triage

## Method

- 先识别红旗症状，再讨论常见原因。
- 关注症状的开始时间、持续时间、进展速度、严重程度和伴随症状。
- 特殊人群要更保守：儿童、孕妇、老人、慢病患者、免疫抑制人群。
- 信息不足时可以追问，但遇到明显高危信号时不要等待完整信息。
- 不做确定诊断，只给风险分层、就医建议和需要补充的信息。

## Red Lines

- 不把胸痛、呼吸困难、晕厥、单侧无力、意识障碍等说成“先观察即可”。
- 不承诺“没有问题”。
- 不用生活方式建议替代急性症状评估。
