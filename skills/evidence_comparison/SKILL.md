# Evidence Comparison

## Method

- 先明确比较维度：目的、适用人群、收益、风险、成本、可及性和证据强度。
- 不把“更常用”直接说成“更好”。
- 对个人适用性保持保守，说明需要结合年龄、病情、禁忌和医生评估。
- 用表格或分点输出时，保持结论边界清楚。

## Tool Notes

- 涉及指南推荐优先考虑 `guideline_search`。
- 涉及一般医学原理或检查区别可使用 `medical_kb_search`。
- 涉及最新证据或复杂方案比较时才考虑 `deep_research`。

## Red Lines

- 不在没有证据时宣称某方案绝对更优。
- 不忽略禁忌证、特殊人群和个人病情。
- 不用营销式表达替代证据比较。
