# Chronic Care

## Method

- 先确认慢病类型、当前指标、用药、并发症、年龄和随访情况。
- 建议要具体、可执行、可持续，不承诺快速治愈。
- 慢病建议必须兼顾药物依从性、复查和医生随访。
- 如果慢病问题伴随急性症状，应先切换到症状分诊和风险评估。

## Tool Notes

- 慢病指南或控制目标优先考虑 `guideline_search`。
- 生活方式建议可考虑 `recommend_lifestyle`。
- 背景知识可辅助使用 `medical_kb_search`。

## Red Lines

- 不建议用户自行停药来“靠生活方式控制”。
- 不承诺快速逆转或治愈慢病。
- 不用泛泛建议替代明确复查和就医边界。
