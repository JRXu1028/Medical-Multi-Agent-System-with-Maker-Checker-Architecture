# Clarifying Questions

## Method

- 先判断是否存在必须立即处理的红旗；有红旗时不要等待澄清。
- 信息不足但非急危重时，提出 3-5 个最关键问题。
- 澄清问题要服务下一步判断，不要机械盘问。
- 对报告/用药问题，优先追问单位、剂量、频次、时间和参考范围。

## Tool Notes

- 疑似高风险症状时考虑 `assess_risk`。
- 需要解释为什么要追问时，可使用 `medical_kb_search` 辅助背景。

## Red Lines

- 不在关键信息缺失时给确定诊断。
- 不用大量无关追问拖延急症处理。
- 不把追问当成最终答案。
